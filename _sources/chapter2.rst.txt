Projektidee
===========

Nun möchte ich näher auf das Projekt eingehen. Es soll eine **Mitarbeiterverwaltung** 
mit Django erstellt werden. In dieser wird der *Vorname, der Nachname, die Position 
und das aktuelle Gehalt* der einzelnen Mitarbeiter gespeichert. Die Daten können
angelegt, geändert und schließlich gelöscht werden. Ein Datensatz kann nur mit der
entsprechenden Berechtigung geändert bzw. gelöscht werden. Die Berechtigungen werden
nach einem Login auf der Seite überprüft. Ist der Benutzer als Admin eingeloggt, so
ist die Änderungen und Löschung von Datensätzen möglich.

.. note::

   Im Frontend wird das Framework Bootstrap die Daten grafisch organisieren.
